# ALX Listing App

## Description
The ALX Listing App is a foundational Airbnb-style listing page built using Next.js, TypeScript, and TailwindCSS. This project focuses on clean architecture, reusable components, and scalable frontend practices.

## Project Structure
- **components/**: Reusable UI components such as Card and Button
- **interfaces/**: Centralized TypeScript interfaces
- **constants/**: Shared constants and configuration values
- **public/assets/**: Images and SVG assets used throughout the app
